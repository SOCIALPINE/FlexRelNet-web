Why Current AI Fails

Built for Hospitals, Not Clinics

Too Large

ResNet-50 needs 97MB. VGG-16 exceeds 500MB. These don't fit on the $35 devices that rural clinics can afford.

Too Dependent

GPU servers, stable electricity, high-bandwidth internet, on-site engineers. Solar clinics with intermittent mobile signal are excluded by design.

Too Expensive

A single screening station costs thousands. For a rural health ministry with 50 clinics to equip, this is not a budget line item ‚Äî it's impossible.

Architecture

How FlexRelNet Works

CNN feature extraction + Sparse Graph Neural Network for relational reasoning

1

CNN Encoder

2 conv layers (16‚Üí32 channels) compress a 28√ó28 image to a 7√ó7 feature map with 49 spatial nodes

2

Sparse Graph Attention

Top-k=8 sparsification retains only the 8 most relevant connections per node (392 of 2,401 possible edges)

3

GNN Reasoning

2 GNN layers propagate context across the sparse graph with residual connections + LayerNorm

4

Classification

Global mean pooling ‚Üí FC(64) ‚Üí Output. Relational patterns preserved through the entire pipeline
