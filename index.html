Problem

Current Medical AI Limitations

Model Size

ResNet-50 requires 97MB (FP32). VGG-16 exceeds 500MB. These sizes prevent deployment on low-cost edge devices.

Infrastructure Requirements

GPU servers cost $5,000+. Requires stable electricity, high-bandwidth internet, and technical staff.

Diagnostic Access Gap

Nearly half the global population lacks access to basic diagnostics. Radiologist ratio below 1:100,000 in many low-income countries.

Lancet Commission on Diagnostics, 2021
